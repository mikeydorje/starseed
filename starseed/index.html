How it works

Each scene is a cloud of particles driven by custom GPU shaders. Drop in an audio file and the Web Audio API splits the signal into 64 frequency bands in real time — those bands push, pull, and reshape the particles every frame.

Eight sliders per scene let you tune the physics: spread, drift, colour, threshold, and more. A narrative arc slowly evolves the parameters over the length of the track so the visuals tell a story even if you never touch a slider.

Behind the scenes, slow drift cycles — long sine waves between 3 and 8 minutes — continually shift rotation and scale. Combined with the arc, this means no two plays look the same.

Hit record and the app renders offline at 60 fps using WebCodecs, muxing straight to MP4 — no server, no upload, everything stays in your browser. Listen mode skips the file and reacts directly to your microphone.

Built with Three.js, WebGL, and zero build tools. Just static files on GitHub Pages.
